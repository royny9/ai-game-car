# 🏎️ Симуляция гоночных машинок с ИИ (Нейросеть + Генетический алгоритм)

Проект представляет собой интерактивную симуляцию на Python, где автономные гоночные машинки учатся проходить трассу. Управление автомобилями полностью возложено на нейросети, которые эволюционируют и обучаются с помощью генетического алгоритма (ГА).

---

## 🚀 Как это работает?

1. **Зрение (Радары):** Каждая машинка выпускает 5 лазерных лучей (вперед, влево и вправо). Лучи измеряют точное расстояние от бампера до белой стены трека.
2. **Мозг (Нейросеть):** Внутри каждой машины работает простая полносвязная нейросеть. Она принимает на вход 5 расстояний от радаров и выдает команды: *«повернуть налево»* или *«повернуть направо»*.
3. **Естественный отбор:** 
   * Поколение из 15 машин запускается со старта.
   * Если машина задевает стену, она погибает.
   * Чем дальше машина проехала по треку, тем выше её показатель успешности (фитнес-функция).
4. **Эволюция:** Когда погибает последняя машина, алгоритм выбирает двух лучших «гонщиков». Их цифровые ДНК (веса нейросети) скрещиваются, слегка мутируют, и создается новое, более умное поколение.

---

## 🛠️ Требования и установка

Для запуска проекта вам понадобятся **Python 3.12+** и библиотека **Pygame-ce** (или Pygame).

1. Установите библиотеку для визуализации:
   ```bash
   pip install pygame-ce
   ```
2. Скачайте код проекта и запустите основной файл (например, переведя код из `.ipynb` в обычный `.py` файл или запустив ячейки в VS Code):
   ```bash
   python main.py
   ```

---

## 📁 Структура кода

* **`NeuralNetwork`** — класс искусственной нейросети. Отвечает за вычисления (прямой проход) и случайные мутации весов.
* **`Car`** — класс гоночного болида. Содержит физику движения, логику работы сенсоров-радаров и проверку столкновений со стенами.
* **`evolve_population`** — ядро генетического алгоритма. Сортирует машины по успеваемости, сохраняет лидеров (элитарность) и создает мутировавших потомков.
* **`main`** — главный игровой цикл, который отрисовывает трассу и перезапускает симуляцию при смене поколений.

---

## 📊 Результаты обучения

* **1–3 поколения:** Машины движутся хаотично и сразу врезаются в ближайший бордюр.
* **5–10 поколения:** Появляются первые особи, способные пройти первый поворот.
* **15–20 поколения:** ИИ полностью понимает геометрию трассы и начинает бесконечно нарезать круги без аварий.
